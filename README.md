## Salon Website
## Description
This is a demo website for a hair-based business
## Features
Responsive navigation bar
Mobile sidebar/hamburger menu
Hero/intro section with background image
About section
Services section
Testimonials slider/carousel
Responsive layout for mobile screens
Footer section
Favicon branding
Hover effects
Sidebar open/close interaction
Image cards
Contact/social links
Mobile-friendly design adjustments
Flexbox layouts
Media queries for responsiveness
Scrollable/swipeable testimonials
Interactive buttons and menus
## Technoligies
 HTML/CSS
 JAVASCRIPT
